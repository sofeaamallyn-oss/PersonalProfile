## Brief Project Description
This project is a web-based Personal Profile Management System developed using Java Servlet, JSP, HTML, CSS and MySQL. The system allows users to create, search, view, edit and delete personal profile records.

## List of Implemented Features

- Add New Profile
- View All Profiles
- Search Profile by Student ID
- Edit Profile Information
- Delete Profile Record
- Store profile data in MySQL database
